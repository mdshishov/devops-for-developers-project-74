## Системные требования
- Git
- Docker
- Make

## Настройка и запуск
```
git clone https://github.com/mdshishov/devops-for-developers-project-74.git

cd devops-for-developers-project-74

make setup

make start
```

### Запуск тестов
```
make test
```
